Guess the Number

Enter a number between 1 and 100

How the Game Works

This is a simple number guessing game. Here's how it works

  1. The computer randomly selects a number between 1 and 100.
  2. You input your guess in the provided text box.
  3. Click the "Guess" button to see if you are correct.
  4. The game will tell you if your guess is too high, too low, or correct.
  5. Keep guessing until you find the correct number.

Code

HTML Structure

The HTML structure defines the layout of the page:

  • <div id="game-container">: This section contains the game elements (input field, button, and message).

CSS for Styling

The CSS part helps style the game:

  • body: Defines the font style for the whole page.
  • #message: Adds margin to space the message from other elements.

JavaScript Logic

The JavaScript part makes the game interactive:

  • Random Number Generation: Math.floor(Math.random() * 100) + 1 generates a number between 1 and 100.
  • checkGuess Function: This function gets the user's guess, compares it to the random number, and updates the message based on whether the guess is too high, too low, or correct.
JavaScript Code Breakdown

Here is a detailed explanation of the JavaScript code:


let randomNumber = Math.floor(Math.random() * 100) + 1;

This line generates a random number between 1 and 100 and stores it in the randomNumber variable.


function checkGuess() {
    const guess = document.getElementById('guess').value;
    const message = document.getElementById('message');

    if (guess == randomNumber) {
        message.textContent = 'Congratulations! You guessed the right number!';
    } else if (guess > randomNumber) {
        message.textContent = 'Too high! Try again.';
    } else {
        message.textContent = 'Too low! Try again.';
    }
}

This function:

  • Gets the user's guess from the input field.
  • Compares the guess with the random number.
  • Updates the message to give feedback based on the comparison.
