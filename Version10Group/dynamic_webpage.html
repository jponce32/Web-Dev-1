Making Your WebPage Dynamic

Written by
Avery R. Johnson


While you can make cool WebPages just using HTML with CSS, there is a whole other world opened up by introducing Javascript into a WebPage. JavaScript is a scripting language designed for the purpose of working with WebPages. It allows you to change CSS and HTML information. In this lesson we will start learning the basics of intergrating JavaScript into your website, making your webpages more dynamic and interactive.


Copy and Paste this code into your body 
<script>
    alert("This is an alert")
</script>
Code File Location
index.html

The code you copy and pasted above executes a pop up alert on your browser. You have probably seen this on websites before, usually an annoying alert, telling you an error or problem. Right now, the code executes this as soon as the html reads the scripted code. This means the alert shows up as soon as the page loads, this is as you can guess, an annoying and not so amazing web design feature. We can fix this however, all we need to do is make our own function that will only send an alert upon a certain condition happening.


Copy and Paste this code into your outside of the script tag 
<button onclick="alert('You Clicked The Button')">
    Press to Alert
</button>
Code File Location
index.html

Result


Now, the alert will only play upon the consumer choosing to do an action that calls the alert code to activate. To end this section off you create a html element that changes upon a criteria being met. If you don't already, create a folder in your project folder titled "JS", then put a file in it called "script.js". This file is for actual javascript code, prior we have been html tags or atrributes that simply call on javascript code, which we are still going to do in this step, but we will now be calling on our own code created within the script.js folder.
